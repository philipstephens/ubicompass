Simple UBI Chart Inspector

How to Use This Tool

This tool helps you inspect the UBI Chart component to verify it's working correctly.

Step 1: Open the UBI Calculator

Make sure the UBI Calculator is running at: http://localhost:5173

Step 2: Open the Browser Console

Open your browser's developer tools by pressing:

  • Chrome/Edge: F12 or Ctrl+Shift+J
  • Firefox: F12 or Ctrl+Shift+K
  • Safari: Cmd+Option+C

Then click on the "Console" tab.

Step 3: Copy and Paste the Script

Copy the entire script below:

// Simple Chart Inspector // Copy this entire script and paste it into your browser console // while viewing the UBI Calculator (function() { console.clear(); console.log('üìä UBI Chart Inspector Running...'); // Find the chart component const purpleHeaders = [...document.querySelectorAll('div[style*="backgroundColor: \\"#673ab7\\""]')]; let chartComponent = null; if (purpleHeaders.length > 0) { chartComponent = purpleHeaders[0].closest('div[style*="borderRadius"]'); console.log('‚úÖ Found chart component via purple header'); } else { console.error('‚ùå Chart component not found!'); return; } // Scroll to the component const rect = chartComponent.getBoundingClientRect(); const scrollTop = window.pageYOffset || document.documentElement.scrollTop; const targetPosition = rect.top + scrollTop - 100; window.scrollTo({ top: targetPosition, behavior: 'smooth' }); // Highlight the component const originalOutline = chartComponent.style.outline; chartComponent.style.outline = '3px solid red'; setTimeout(() => { chartComponent.style.outline = originalOutline; }, 2000); // Check if we're in chart or table view const chartBars = [ ...chartComponent.querySelectorAll('div[style*="height"][style*="%"]'), ...chartComponent.querySelectorAll('div.bg-blue-500'), ...chartComponent.querySelectorAll('div.bg-green-500') ]; const hasTable = chartComponent.querySelector('table') !== null; const isChartView = chartBars.length > 0 && !hasTable; console.log(`üìà Current view: ${isChartView ? 'Chart View' : 'Table View'}`); console.log(`üßÆ Chart bars found: ${chartBars.length}`); // Find the toggle button const toggleButtons = [ ...document.querySelectorAll('button svg[viewBox="0 0 20 20"]'), ...document.querySelectorAll('button:has(svg)'), ...document.querySelectorAll('span.chart-icon-container') ]; if (toggleButtons.length > 0) { const toggleButton = toggleButtons[0].closest('button') || toggleButtons[0]; console.log('‚úÖ Found toggle button'); // Ask if user wants to click the toggle button if (confirm('Do you want to toggle between chart and table view?')) { toggleButton.click(); console.log('‚úÖ Clicked toggle button'); } } else { console.log('‚ùå Toggle button not found'); } console.log('üèÅ Chart inspection complete'); })();

Step 4: Paste and Run in Console

Paste the script into your browser console and press Enter to run it.

The script will:

  • Find the chart component
  • Scroll to it
  • Highlight it briefly with a red outline
  • Tell you if it's in chart or table view
  • Count the chart bars
  • Find the toggle button
  • Ask if you want to toggle between views

Troubleshooting

If the script doesn't work:

  • Make sure the UBI Calculator is running at http://localhost:5173
  • Check that JavaScript is enabled in your browser
  • Try refreshing the page before running the script
  • Check the browser console for any errors
