‚ùì Help & Troubleshooting

If you're having issues with the chart inspection:

  • Make sure the server is running - The UBI Calculator must be accessible at http://localhost:5173
  • Check browser console - Open your browser's developer tools (F12) to see any errors
  • Allow popups - This tool opens a new tab, so make sure popups are allowed
  • Refresh the page - If the chart doesn't load, try refreshing the UBI Calculator page

For more detailed debugging, you can:

  • Open the browser console in the UBI Calculator tab
  • Run localStorage.setItem('debug', 'true') to enable debug mode
  • Refresh the page to see more detailed logging
